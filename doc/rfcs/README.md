# RFCs

1. [Procedures Framework](procedures-framework.md) - neutral `doc/procedures/` framework and its approval process.

This directory stores Request for Comments (RFC) documents for architectural or design decisions with future impact.

Use this directory when a task requires a new design decision before implementation.

- Create RFCs from the template in `doc/rfcs/rfc.template.md`.
- Link relevant RFCs from task pages under `todos/`.
